## Важно

English version: [README-EN.md](README-EN.md)

Версия описания задания: 1

Используйте этот репозиторий как шаблон структуры решения.

## Что фиксируется как решение

Проверяется состояние последнего коммита в ветке `main`.

## Минимальные требования к структуре

В репозитории должны присутствовать:

1. `Dockerfile`
2. `docker-compose.yml`
3. `dvc.yaml`
4. `params.yaml`

Именно это проверяет примерный `.gitlab-ci.yml` в этом шаблоне.

## Важно про запуск

Во время проверки ожидается, что сервис после `docker compose up -d`:

1. слушает на порту `80`;
2. отвечает `200` на `GET /ready` в пределах 180 секунд;
3. реализует эндпоинты согласно `openapi.yml`.

## Академическая честность

При использовании внешнего кода указывайте источник. Работы проверяются на неправомерные заимствования.
